# Actionable PR report

Prioritize blocked release work, failing required checks, requested author
changes, and pending required reviews. Keep ready-to-merge items visible but
separate from work that needs intervention.

Each action item should identify the pull request, the observed status, the next
action, and the responsible role or person when known.
