Model resolution precedence

How the runtime picks the model for a given call, highest first:

  1. Explicit requested model (agent frontmatter model:)
  2. Provider env (FOUNDRY_MODEL / AZURE_OPENAI_DEPLOYMENT)
  3. AZURE_FUNCTIONS_AGENTS_MODEL
  4. Provider default

Auto-detection order

When AZURE_FUNCTIONS_AGENTS_PROVIDER is unset:

1 · AZURE_OPENAI_ENDPOINT 2 · FOUNDRY_PROJECT_ENDPOINT 3 · OPENAI_API_KEY
🔒 Secrets (*_API_KEY) are write-only in the portal and stored in the secret store. Prefer identity-based auth where possible.
